# Weather Website

Weather Website is a responsive web application designed to display real-time weather information through a simple and interactive interface. The project focuses on providing weather updates with a clean design and dynamic user experience.

## Features

* Real-time weather information
* Responsive web design
* Interactive user interface
* Dynamic weather updates
* Clean and user-friendly layout

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Purpose

The project was developed to create a simple and responsive weather application capable of displaying weather information in an interactive and visually appealing way.
