# 🌦️ Neo Weather Dashboard

A futuristic and visually stunning **Weather Report Website** built using **HTML, CSS, and JavaScript** with real-time weather data integration using the WeatherAPI.

The project features a high-tech glassmorphism dashboard, animated weather effects, dynamic themes, neon visuals, and a responsive modern UI.

---

# 🚀 Features

✨ Real-Time Weather Data
✨ Search Weather by City Name
✨ Futuristic Wine-Themed Dashboard
✨ Glassmorphism UI Design
✨ Dynamic Weather Animations
✨ Rain & Snow Particle Effects
✨ Responsive Design for All Devices
✨ Interactive Hover Effects
✨ Live Temperature, Humidity, Wind & Pressure
✨ Dynamic Background Themes Based on Weather Conditions

# 🌍 How It Works

1. User enters a city name
2. The app fetches real-time weather data using WeatherAPI
3. Dashboard updates dynamically
4. Weather-based effects are applied:

   * 🌧️ Rain Animation
   * ❄️ Snowfall Effect
   * ☀️ Sunny Glow Theme
